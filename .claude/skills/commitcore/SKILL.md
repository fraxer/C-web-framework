---
name: commitcore
description: Commits changes in submodule backend/core to git with an automatically generated message
---

Instructions:
You must create a commit with the files already added to staging.
1. View the staged changes with git diff --cached
2. Analyze the changes and create a commit message in English
3. Create a commit with a commit message

Message format:
<type>: <short description>

Long description (if needed)

Types: feat, fix, refactor, docs, style, test, chore

Types without specifications/parentheses

Do not mention yourself as a co-author
